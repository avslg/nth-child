CSS селектор div:nth-child()

Правила

CSS псевдокласс :nth-child() находит один или более элементов, основываясь на их позиции среди группы соседних элементов.

/* Выбирает каждый четвёртый элемент среди любой группы соседних элементов */
:nth-child(4n) { color: lime;
}

Описывает элементы среди группы соседних с номерами, соответствующими паттерну An+B, где n — любое положительное целое число или ноль. Нумерация элементов начинается с единицы. Значения A и B должны быть целые числа.

/* Выбирает an+b */
:nth-child(an+b) { color: lime;
}

Примеры

:nth-child(3n+4)

Выбирает элементы с номерами
4, 7, 10, 13, 16 и тд

:nth-child(odd)

Выбирает элементы с номерами
1, 3, 5 и тд

:nth-child(even)

Выбирает элементы с номерами
2, 4, 6 и тд

:nth-child(2)

Выбирает элемент 2
